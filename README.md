
# VoxelCraft

My attempt to write a Minecraft clone in C++.<br>
Heavily influenced by GamesWithGabe, SolvingCode, CZDanol, Hopson and others.


#### Using
- VSCode
- g++
- GLFW
- GLADE
- stb
